# 👕 Outfitly E-commerce

**Outfitly E-commerce** is a simple web-based e-commerce application developed using **PHP, HTML, CSS, and JavaScript**.  
This project was created as part of a **Web Technology lab practice** to demonstrate the basic structure of an online shopping website.

The application allows users to **register, log in, and access a dashboard**, while showcasing outfit products through a structured web interface.

---

##  Features

- User registration system
- User login authentication
- Dashboard for logged-in users
- Reusable header and footer components
- Organized project structure
- Static product/content display
- Basic frontend styling with CSS and JavaScript

---

##  Technologies Used

- **PHP** – Backend logic
- **HTML5** – Page structure
- **CSS3** – Styling
- **JavaScript** – Basic interactivity
- **XAMPP / Apache Server** – Local development server

---

### File Description

| File | Description |
|-----|-------------|
| `index.php` | Main homepage of the website |
| `header.php` | Reusable header section |
| `footer.php` | Reusable footer section |
| `login.php` | User login page |
| `register.php` | User registration page |
| `login_process.php` | Handles login authentication |
| `dashboard.php` | User dashboard after login |
| `content.php` | Displays main page content |
| `assets/` | Contains images, stylesheets, and JavaScript |
